# Teachable-Machine
A machine learning image classification project built using Google Teachable Machine and exported as a TensorFlow.js model. This model classifies images into two categories:

🌭 Hot Dog
🚫 Not Hot Dog
📌 Project Overview

This project uses a pre-trained deep learning model from Teachable Machine to identify whether an uploaded image contains a hot dog or not.

The model was trained using custom image datasets and exported in TensorFlow.js format, making it easy to integrate into web applications.
